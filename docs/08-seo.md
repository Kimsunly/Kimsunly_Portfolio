# SEO Philosophy

The portfolio is more than a personal website.

It is a professional engineering profile.

SEO should support discoverability while maintaining a natural reading experience.

Avoid keyword stuffing.

Write for humans first.

Optimize for search engines second.

Optimize for AI understanding third.

The content should accurately represent technical skills without exaggeration.
